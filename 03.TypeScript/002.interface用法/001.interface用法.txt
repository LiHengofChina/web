

//================================= interface
//================================= interface

在 TypeScript 中，interface 用来定义对象的 "结构" 或 "形状"，
本质上就是："这个对象应该有哪些属性、类型是什么" 。

//================================= 歧义
//================================= 歧义
interface 是 "定义接口"

虽然我们叫它 "接口"，但它描述的内容就是一个对象的结构。

//================================= 举个比喻
//================================= 举个比喻

interface Person {
  name: string;
  age: number;
}

	//"我需要一个 对象，里面必须有 name 和 age 这两个字段，类型必须正确。"
