TypeScript（TS）是 JavaScript 的一个超集，
	它是在 ES6（甚至更高版本）基础上增加了 "类型系统" 和 "其他功能的语言"。

//========================================= ES6 (ECMAScript 2015)
//========================================= ES6 (ECMAScript 2015)

		//是 JavaScript 的一种规范版本（新增了 let/const、箭头函数、Promise、class 等）

//========================================= TypeScript
//========================================= TypeScript

是 JavaScript 的超集，加了类型检查、接口、泛型、装饰器等功能，编译后变成纯 JavaScript

//========================================= TypeScript 的核心价值：重点（在开发阶段发现错误）
//========================================= TypeScript 的核心价值：重点（在开发阶段发现错误）

	（1）静态类型检查写代码时就能发现类型错误，不必等运行时报错
	（2）更强的 IDE 支持	自动补全、跳转、提示都更强
	（3）适合中大型项目	代码更可靠、更易维护

//========================================= 总结
//========================================= 总结

TypeScript 是一种带有 "类型" 的 JavaScript，能提前发现错误，提高代码质量。
你完全可以把它当成 "更安全的 JavaScript"。

你熟悉 ES6 的话，学 TypeScript 非常顺，主要多加几个类型关键字，比如 number、string、boolean、interface 等
