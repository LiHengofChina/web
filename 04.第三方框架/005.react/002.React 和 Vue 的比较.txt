

对比点			React							Vue
————————————————————————————————————————————————————————————————————————
定位			UI 库							完整框架
语法风格		JSX（HTML 写在 JS 里）			模板语法（HTML、JS、CSS 分离）
组件写法		函数组件 + Hooks 为主			<template> + script + style 结构
数据绑定		单向绑定（props）+ useState		单向 + 双向（v-model）
状态管理		自带轻量管理（useState） 		常用 Redux / Zustand 等	Vuex / Pinia
生命周期		useEffect 替代生命周期钩子		created、mounted 等
指令系统		没有内置指令系统				有丰富指令（v-if、v-for、v-bind 等）
学习曲线		灵活但偏自由，生态多样需组合	结构清晰、上手快


//———————————————————————————————————————————————————————————————————————— 相似点：
//———————————————————————————————————————————————————————————————————————— 相似点：
（1）都是组件化 + 虚拟 DOM + 响应式更新。
（2）都支持开发 SPA 应用。
（3）都有构建工具（Vite、Webpack）。


//———————————————————————————————————————————————————————————————————————— 不同点：
//———————————————————————————————————————————————————————————————————————— 不同点：
（1）React 更灵活但更碎片化（什么都要自己选）。
（2）Vue 更集成化，上手快、适合中小团队快速开发。
（3）JSX 对 Vue 用户来说初看可能有些不习惯（逻辑和视图混在一起写）。
