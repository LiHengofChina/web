

//==================================================  React 简介
//==================================================  React 简介

	React 是 Facebook 推出的 前端 UI 库（注意它自称是“库”而不是完整框架），
	用于构建用户界面，特别是构建组件化的单页应用（SPA）。


//=============================== React 特点：
//=============================== React 特点：
	（1）JSX 语法：看起来像 HTML，但实际上是 JavaScript，标签写在 JS 中。
	（2）函数组件 + Hooks 是主流写法
		 （React 18 基本都是函数组件 + useState、useEffect 等）。
	（3）虚拟 DOM：和 Vue 一样，使用虚拟 DOM 实现高性能渲染。
	（4）单向数据流：数据从父组件传子组件，推荐用 props 和 state 管理状态。
	（5）生态灵活：React 本身只管视图，路由（react-router）、
		 状态管理（Redux、Recoil）、请求（axios/fetch）都需自己组合。
