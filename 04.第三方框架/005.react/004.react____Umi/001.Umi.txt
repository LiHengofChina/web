//======================================= 什么是 Umi？

Umi 是阿里巴巴开源的企业级前端框架，
用于快速搭建 React 应用。

Umi 是阿里巴巴开源的企业级前端框架，用于快速搭建 React 应用。
它是 基于 React 封装的一整套解决方案，集成了路由、构建、Mock、权限控制、国际化等。

//======================================= Umi 的特点：
//======================================= Umi 的特点：

✔ 基于 React						使用 React 写页面
✔ 内置路由系统						约定式路由，按文件夹自动生成页面路由
✔ 插件机制强						功能像搭积木，可扩展性好
✔ 支持 TypeScript					默认支持
✔ 支持微前端						内置 qiankun 支持
✔ 支持 SSR							可服务端渲染（可选）
✔ 与 Ant Design Pro 深度集成		企业后台模板用得多
