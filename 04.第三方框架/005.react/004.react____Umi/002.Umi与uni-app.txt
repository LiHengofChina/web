

//=========================================================================

Umi				//基于 React 的前端 Web 框架，主攻 企业后台系统（如管理平台）

uni-app			//基于 Vue 的跨平台开发框架，主攻 小程序 + App + H5 多端开发



//======================== 类比说：
//======================== 类比说：

🏢 Umi 更像 Vue + Vue Router + Webpack + CLI 的组合，但基于 React。
📱 uni-app 更像是“Vue 写一次，多个平台运行”的跨平台框架。

//======================== 总结一句话：
//======================== 总结一句话：

Umi 是写 Web 后台用的 React 框架，
uni-app 是写小程序和 App 的 Vue 跨平台框架，功能方向完全不同。

你写企业后台用 Umi，写小程序/移动端用 uni-app。
