	}

// count_1、count_2是一个计算属性，
//它的定义方式是通过一个函数实现的。

//在 Vue 中，计算属性是以函数的形式定义的，

但是它们在使用时表现为 "属性"。

<p>Count: {{ count_1 }}</p>
<p>Double Count: {{ count_2 }}</p>

计算属性在 Vue 组件中非常有用，
它们能够帮助你处理复杂的逻辑并保持视图和数据的同步，
具有缓存特性且依赖更新时自动重新计算。
